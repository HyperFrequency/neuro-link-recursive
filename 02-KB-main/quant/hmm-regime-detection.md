# HMM Regime Detection in Finance

## Overview

[[Hidden Markov models]] (HMMs) model asset returns as generated by a discrete latent state $$S_t$$ evolving as a Markov chain, with observed return $$r_t | S_t$$ drawn from a state-conditional emission distribution. They provide unsupervised [[regime detection]] — uncovering bull/bear, low-vol/high-vol, or liquidity states.

## Conceptual Model

An HMM is $$(A, B, \pi)$$: transition matrix $$A_{ij} = P(S_{t+1}=j | S_t=i)$$, emission $$B_i(r_t) = p(r_t | S_t=i)$$, initial $$\pi$$. For Gaussian HMM, $$B_i = \mathcal{N}(\mu_i, \sigma_i^2)$$ with state-specific mean/variance. Inference via [[forward-backward]] for filtering/smoothing, [[Viterbi]] for MAP decoding, [[Baum-Welch]] (EM) for parameter fitting.

## Details

Classical application: Hamilton (1989) introduced the 2-state regime-switching model for GNP growth. In finance, Ang-Bekaert (2002) and Ryden-Terasvirta-Asbrink (1998) fit HMMs to equity returns, identifying a low-vol high-return state and high-vol low-return state — anticipating the [[volatility-return]] asymmetry later formalised in [[stochastic volatility]] models.

[[State selection]]: AIC, BIC, cross-validated log-likelihood, or DIC. Over-fitting common with high state count (curse of local optima). Practitioners often fix 2-3 states based on economic interpretability.

**Predictive usage**: filtered probabilities $$P(S_t = i | r_{1:t})$$ can signal regime shifts in real time — e.g., increasing probability of high-vol state triggers position de-risking. [[Markov-switching GARCH]] extends to state-dependent conditional variance. [[HMM-VAR]] combines with vector autoregression for multi-asset regime detection.

**Pitfalls**: HMMs assume stationary transition matrix — fails through structural breaks. [[Baum-Welch]] converges to local optima; multiple random inits recommended. States can be label-swapped between training runs (identifiability). [[Reinforced]] HMMs allow semi-Markov durations (HSMM) to fix exponential-dwell-time misfit.

**Modern alternatives**: [[change-point detection (CUSUM + PELT)]] for agnostic break detection; [[neural]] state-space models (SNP, DSSM); [[regime-switching copulas]] for dependence structure. [[Structural break tests (Chow + Bai-Perron)]] provide frequentist complements.

Libraries: `hmmlearn` (Python), `depmixS4` (R), `Pomegranate`.
