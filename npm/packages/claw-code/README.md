# @neuro-link/claw-code

neuro-link helper for Claw-Code. Installs hook scripts into `~/.claw/hooks/neuro-link/` and prints proxy env vars.

## Usage

```bash
npx @neuro-link/claw-code install
npx @neuro-link/claw-code status
npx @neuro-link/claw-code uninstall
```

Export after install:

```bash
export ANTHROPIC_BASE_URL="http://127.0.0.1:7414"
export NLR_ROOT="$HOME/Desktop/HyperFrequency/neuro-link"
```
