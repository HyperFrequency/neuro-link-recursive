# @neuro-link/forge-code

neuro-link helper for ForgeCode. Installs hook scripts into `~/.forge/hooks/neuro-link/` and prints proxy env vars.

## Usage

```bash
npx @neuro-link/forge-code install
npx @neuro-link/forge-code status
npx @neuro-link/forge-code uninstall
```

Export after install:

```bash
export ANTHROPIC_BASE_URL="http://127.0.0.1:7414"
export NLR_ROOT="$HOME/Desktop/HyperFrequency/neuro-link"
```
