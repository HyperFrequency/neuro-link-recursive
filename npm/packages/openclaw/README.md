# @neuro-link/openclaw

neuro-link helper for OpenClaw. Installs hook scripts into `~/.openclaw/hooks/neuro-link/` and prints proxy env vars.

## Usage

```bash
npx @neuro-link/openclaw install
npx @neuro-link/openclaw status
npx @neuro-link/openclaw uninstall
```

Export after install:

```bash
export ANTHROPIC_BASE_URL="http://127.0.0.1:7414"
export NLR_ROOT="$HOME/Desktop/HyperFrequency/neuro-link"
```
